DbUnit
------

DbUnit是一个JUnit扩展,针对数据库驱动的项目.
DbUnitd支持数据库中的数据导出和导入和XML数据集的能力。从2.0版本开始，DbUnit也支持非常大的数据集流模式下使用。 DbUnit的还可以帮助你验证数据是否符合预期

##Maven依赖

```bash
    <dependency>
      <groupId>org.dbunit</groupId>
      <artifactId>dbunit</artifactId>
      <version>${dbunitVersion}</version>
      <scope>test</scope>
    </dependency>
```
